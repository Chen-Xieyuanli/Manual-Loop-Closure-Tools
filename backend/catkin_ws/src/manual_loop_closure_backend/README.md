# manual_loop_closure_backend

Catkin package for the offline optimizer used by the GUI.

GUI 使用的离线优化器 catkin 包。

Build:

```bash
cd ~/my_git/Mannual-Loop-Closure-Tools
bash scripts/build_backend_catkin.sh
```

Binary:

```bash
backend/catkin_ws/devel/lib/manual_loop_closure_backend/manual_loop_optimize
```
